CORS Troubleshooting

  1. Verify that your Appwrite project has the following domains added in the console:
    • http://localhost:3000
    • http://192.168.1.7:3000
  2. Check that your .env.local file has the correct values for:
    • NEXT_PUBLIC_APPWRITE_ENDPOINT
    • NEXT_PUBLIC_APPWRITE_PROJECT_ID
    • NEXT_PUBLIC_APPWRITE_HOSTNAME
  3. Verify that client.ts is correctly setting the Origin header for CORS requests
  4. Try clearing browser cache or using private browsing
  5. Check browser console for detailed error messages
